ACCESS MODIFIERS

Public - accessible anywhere
Protected - within the same class, package, and child classes
Default - within the same class and package
Private - only within the same class



Constructors are not methods (functions)
Encapsulation - getting and setting private properties/members/fields/states considering data validation


Association - relationships build between objects of different types. There are two types of association:

AGGREGATION: in this type of relationships the objects involved do not necessarily need each other to survive.
COMPOSITION: in this type of relationships the objects involved strongly dependent on each other, one cannot survive without the other.
